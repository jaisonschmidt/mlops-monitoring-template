# Screenshots dos Dashboards Grafana

Este diretório contém screenshots dos dashboards para referência e documentação.

## Como Capturar Screenshots

Após configurar e executar o sistema:

1. Acesse Grafana em `http://localhost:3000`
2. Navegue até cada dashboard
3. Capture screenshots e salve aqui com os nomes:
   - `overview.png` - Dashboard System Overview
   - `api-health.png` - Dashboard API Health & Performance
   - `ml-metrics.png` - Dashboard ML Model Metrics
   - `business-churn.png` - Dashboard Business Intelligence - Churn

## Formato Recomendado

- **Resolução**: 1920x1080 (Full HD)
- **Formato**: PNG
- **Modo**: Com dados reais após executar teste de carga
